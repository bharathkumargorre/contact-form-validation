# Contact Form Validation Using JavaScript

## 📌 Project Overview

This project is a responsive Contact Form built using HTML, CSS, and JavaScript. It demonstrates client-side form validation by checking user inputs before submission and providing real-time feedback through error and success messages.

## 🚀 Features

* Responsive and user-friendly design
* Name field validation
* Email field validation using Regular Expressions (Regex)
* Message field validation
* Dynamic error messages
* Success message on valid submission
* Prevention of invalid form submission using JavaScript

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
